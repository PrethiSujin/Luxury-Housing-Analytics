# 🏡 Luxury Housing Analytics Dashboard – Bangalore

## 📌 Overview

This project analyzes luxury housing trends in Bangalore using Power BI, Python, and SQL. It provides insights into market demand, builder performance, buyer behavior, and geographical distribution.

---

## 🎯 Key Objectives

* Identify top-performing builders
* Analyze booking trends across quarters
* Understand buyer segmentation
* Discover high-demand micro-markets

---

## 📊 Dashboard Features

* 💰 Revenue KPI
* 📊 Booking Count KPI
* 💎 Revenue per Booking
* 📈 Revenue Growth Analysis
* 🏆 Top Builders by Revenue
* 🗺️ Geographical Insights (Map)
* 📉 Market Trends by Quarter
* 🏠 Configuration Demand Analysis

---

## 📈 Key Insights

* East Bangalore dominates luxury housing demand
* Top builders contribute similar revenue share
* Revenue growth shows slight fluctuations across quarters
* Demand is evenly distributed across configurations

---
## 🧹 Data Cleaning (Python)

Data preprocessing and cleaning were performed using Python (Pandas), including:

* Handling missing values
* Removing inconsistencies
* Feature engineering
* Exporting cleaned dataset for Power BI
